# Androxine v2 WebApp

Минимальный Telegram WebApp для работы с тренировками.

## Запуск

1. Установить зависимости:
   - npm install
2. Запустить dev-сервер:
   - npm run dev

## Что есть

- Вкладки: Категории, Упражнения, Шаблоны, Подходы
- Базовые заглушки под REST API
- MUI интерфейс с верхними и нижними вкладками
